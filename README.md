# Clinical Expert System using Prolog

## Description
A rule-based AI system that diagnoses diseases based on symptoms using Prolog.

## Features
- Symptom-based diagnosis
- Multiple disease detection
- Provides medical advice

## How to Run
1. Install SWI-Prolog
2. Load file:
   [clinicalsector].
3. Run:
   start.

## Technologies
- Prolog (SWI-Prolog)
